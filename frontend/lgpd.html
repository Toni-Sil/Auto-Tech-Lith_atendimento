1. Controlador de Dados

Auto Tech Lith atua como Controladora dos dados pessoais coletados no âmbito da prestação de serviços de automação de atendimento, conforme definição da LGPD (Art. 5º, VI).

Quando o cliente utiliza a plataforma para processar dados de seus próprios usuários finais (ex.: conversas de clientes no WhatsApp), o cliente atua como Controlador e a Auto Tech Lith como Operadora (Art. 5º, VII), processando dados estritamente conforme as instruções do cliente.

2. Base Legal para Tratamento

Tratamos dados pessoais com base nas seguintes hipóteses legais previstas na LGPD (Art. 7º):

  • Execução de contrato — dados necessários para prestação dos serviços contratados
  • Legítimo interesse — analytics de uso, melhoria da plataforma, segurança
  • Consentimento — comunicações de marketing (opcional, revogável a qualquer momento)
  • Cumprimento de obrigação legal — quando exigido por autoridade competente

3. Categorias de Dados Tratados

Dados dos Clientes (contratantes da plataforma)

  • Identificação: nome, e-mail, telefone, CNPJ/CPF
  • Credenciais de acesso (senha com hash, token MFA)
  • Dados de uso e faturamento
  • Chaves de API (criptografadas com Fernet)

Dados dos Usuários Finais (clientes dos clientes)

  • Mensagens trocadas com os agentes de IA via WhatsApp, Telegram ou Web Chat
  • Número de telefone ou identificador de canal
  • Dados fornecidos voluntariamente durante a conversa

Importante: A Auto Tech Lith não coleta dados sensíveis (Art. 5º, II) de forma intencional. O cliente é responsável por não direcionar dados sensíveis às conversas dos agentes sem as devidas salvaguardas.

4. Transferência Internacional de Dados

Alguns dados podem ser processados por provedores de IA localizados fora do Brasil (ex.: OpenAI nos EUA), conforme as chaves configuradas pelo próprio cliente. Estas transferências ocorrem com base no Art. 33 da LGPD, sob cláusulas contratuais adequadas. O cliente é responsável por informar seus usuários finais sobre essa possibilidade.

5. Direitos dos Titulares

Conforme Art. 18 da LGPD, os titulares têm direito a:

  • ✅ Confirmação da existência de tratamento
  • ✅ Acesso aos dados pessoais
  • ✅ Correção de dados incompletos, inexatos ou desatualizados
  • ✅ Anonimização, bloqueio ou eliminação de dados desnecessários
  • ✅ Portabilidade dos dados a outro fornecedor
  • ✅ Eliminação dos dados tratados com base no consentimento
  • ✅ Informação sobre compartilhamento com terceiros
  • ✅ Revogação do consentimento
  • ✅ Petição à ANPD (Autoridade Nacional de Proteção de Dados)

Para exercer qualquer direito, entre em contato pelos canais abaixo. Prazo de resposta: até 15 dias úteis.

6. Medidas de Segurança Implementadas

  • 🔐 Criptografia de chaves de API com Fernet (AES-128-CBC)
  • 🔑 Senhas com hash bcrypt + salt automático
  • 🛡️ Autenticação multifator (MFA/TOTP) disponível para todas as contas
  • 🔒 Comunicação exclusivamente via HTTPS/TLS
  • 🏗️ Arquitetura multi-tenant com isolamento completo por tenant_id
  • 📊 Logs de acesso e auditoria internos
  • ⏱️ Tokens JWT com expiração e revogação por logout

8. Incidentes de Segurança

Em caso de incidente de segurança que possa causar risco ou dano relevante aos titulares, notificaremos a ANPD e os titulares afetados no prazo de 72 horas após a confirmação do incidente, conforme Art. 48 da LGPD.
