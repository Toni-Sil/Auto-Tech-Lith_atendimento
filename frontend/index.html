🔗 Webhooks & APIs

🔔
O que é um Webhook?
Um Webhook (Saída) é uma notificação automática que este sistema envia para você quando algo acontece — como uma nova mensagem, reunião agendada ou ticket aberto. Você configura a URL do seu sistema e nós chamamos ela com os dados do evento.

Exemplo: "Quando um lead novo entrar, avise meu CRM em https://meucrm.com/webhook"
Nome URL de Destino Método Status Último Teste Ações
🔌
O que é uma API de Integração?
Uma API (Entrada) é um endpoint que sistemas externos chamam para enviar dados a este agente. Ao registrar uma integração aqui, você documenta de onde vêm os dados — como um CRM, formulário do site, ou plataforma de e-commerce.

Exemplo: "Meu formulário do site envia o lead para https://meuagente.com/api/webhook — registre aqui para controlar esse fluxo."
